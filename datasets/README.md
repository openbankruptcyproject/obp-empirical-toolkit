# Datasets

Versioned, citable datasets published by the Open Bankruptcy Project.
All datasets are released under Creative Commons Attribution 4.0
(CC BY 4.0).

## Currently published

- `subv_quarterly.csv` &mdash; Quarterly Subchapter V filing volume,
  2020-Q1 through 2025-Q3. Source: Administrative Office of the U.S.
  Courts, Table F-2A.

## Citation format

```
Open Bankruptcy Project (2026). [Dataset name], v[N].
501(c)(3) public charity (EIN 41-5159631).
URL: https://github.com/openbankruptcyproject/obp-empirical-toolkit/blob/main/datasets/[file]
License: CC BY 4.0
```

## Versioning

Each dataset is tagged in the repo with a release indicating data cutoff.
Methodology for each dataset is documented in `methodology/`.

## Data quality

These datasets reflect best-available public data. Specific limitations are
documented in the methodology files. Always cross-reference with primary
sources for citation-grade work.
