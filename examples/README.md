# Examples

Reproducible analysis examples that demonstrate the toolkit's use.

## Files

- `reproduce_subv_curve.py` &mdash; loads `datasets/subv_quarterly.csv` and
  produces the SVG chart embedded in
  https://viz.openbankruptcyproject.org/subv-adoption-curve/
- `classify_aps_demo.py` &mdash; demonstrates the AP outcome classifier
  against synthetic test data

## Usage

```bash
python examples/reproduce_subv_curve.py
python examples/classify_aps_demo.py
```
